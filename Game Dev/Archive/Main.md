The purpose of this file is to gather all inputs for the game. It is not very structured, but it is meant to be sent to Copilot Research Agent to compile into a comprehensive Developer Document that will define all aspects of the game. This is a total braindump.

## Campaign Map

- **Starting Point in Midgard**: The game begins in Midgard, the world of humans, which serves as the central hub.
- **Explore Other Realms**: From Midgard, the player travels to different realms, such as Jotunheim, Asgard, etc.

- **Shared Structure**: Villages, caves, and other elements exist in all realms, but with unique themes for each one.
- **Interaction and Resources**: The player can interact with different beings and gather resources specific to each realm.

- **Dynamic World Exploration**: Each realm offers a dynamic environment where the player’s movement and choices impact the discovery of artefacts.
- **Artefact Acquisition**: Artefacts become accessible as players explore further, encouraging thorough exploration.
- **Local vs. Main Objectives**: While there’s an overarching main quest (e.g., collecting artefacts), each realm includes local missions that add depth and reward further exploration.
- **Player Choice**: Players can choose to focus on the main quest quickly or delve deeper into each realm to gain additional items, knowledge, and character progression.
- **Balanced Challenge**: The system ensures that exploration is rewarding without making it too easy, maintaining excitement and engagement.
