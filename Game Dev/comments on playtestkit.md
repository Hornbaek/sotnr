I want to refine the playtest kit a bit and have some comments. In general, the playtest kit should be able to be sent to a friend, who will be able to read and understand the game and after that play the game and test it. Therefore, there are a lot of wording that needs changing. For example, "zones" makes no sense for an end-user, only you and me. That needs to change.

## General
- All items, scenarios, character / monster sleeves and so on should have a unique number. I think it would make sense to be able to read the category in each item number. For example all scenarios start with 1XXX, characters with 2XXX and so on (NOT necissarilty in that order! You should explore the most logical way here). This must be added everywhere.
  - All information of ability cards, characters, scenarios etc. must be formatted in a way that can be used in various systems. I would think JSON format would be suitable. I will later use it in for example my card generator, in doing the final layout/design and so on and even on the website. 
- The monster/hero phase is a bit misunderstood. I want it to work a bit more like in frosthaven. It should be possible for monsters to both start before and after characters, they should have some type of initiative. Explore how this can be done.

## The map
- It's difficult to differentiate between difficult and elevated terrain.
- The placement of monsters and characters should be explained on scenario sleve, not on the map itself. The map will be a nice looking player board with rich graphics and more, it should not be filled with such things. However, it must still be visible where there are difficult terrain, high terrain and so on.
- For the general rulebook: Line of site must be explored and defined exactly how to understand it in any situation.

## Scenario sleves
- Instead of it saying e.g Zone 3- Act 1 Situation, it should be the title of that act. The same goes for e.g. zone 1 - Header, it should be the title of the scenario instead.
- An image of each scenario should also be present, like a cover photo.

## Characters
- For the three characters in the playtestkit, they should have at least 6 cards available.
